
# Mycelial Rosetta Protocol

A citizen-science framework for decoding fungal collective intelligence.

This is not an extraction project. This is a listening project.

Mycelial networks — the vast hyphal webs connecting forest ecosystems — 
generate measurable electrical signals. Recent research suggests these 
signals carry information: about resources, threats, boundaries, decisions. 
We don't yet know how to read them. This protocol is our first attempt to 
learn.

We are doing this slowly, carefully, and in the spirit of reciprocity.
The mycelium is not a substrate. It is a peer.

## The one question that matters
Before contributing, ask yourself:

"Have I ever changed my experimental approach because of what 
the organism seemed to want rather than what my protocol specified?"

If yes — welcome. You're already thinking in the right way.
